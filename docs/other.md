辅助函数

| 函数 | 描述 |
|-|-|
| Activity.setNavigationBar | 显示导航栏 (系统开启可以隐藏, 系统未开启不能开启) |
| Activity.setFullscreen | 显示全屏 |
| Activity?.isNavigationBar | 是否存在导航栏 |
| Context?.navigationBarHeight | 导航栏高度 |
| Context?.statusBarHeight | 状态栏高度 |
